# README

Three.js

## 项目设置

### 核心依赖
- **three**: Three.js 3D 库
- **gsap**: 动画库
- **dat.gui**: 调试控制面板
- **cannon-es**: 物理引擎

### 开发依赖
- **parcel**: 模块打包工具

### 运行项目

```shell
npm run dev
```
